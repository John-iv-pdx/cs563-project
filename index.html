Multi-Layer Perceptron

 I developed this program as a part of the Machine Learning course I took at Portland State University. This application is a build-on of the simple perceptron. The data this program ingests is called the MNIST data set. It is a data set of 28x28 handwritten digits. The objective of this program is to train the MLP to accurately identify the number being written when provided it as an input.
  • C++ based application.
  • Heavy algorithm-to-code demonstration using stochastic gradient descent (SGD) and Backpropagation.
  • Accuracy ~95% on the MNIST dataset.
  • Basic principles of perceptrons, and the learning algorithm for weight training.
  • OOP design structure.
